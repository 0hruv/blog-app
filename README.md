# Full-Stack Blog with Auth

This project was developed as part of my **internship**.

A comprehensive full-stack blog application featuring user authentication, post creation, and management.

## Tech Stack
- **Frontend:** React, TypeScript, Vite, Vanilla CSS
- **Backend:** Node.js, Express
- **Database:** SQLite
- **Auth:** JWT (JSON Web Tokens) & bcrypt

## Features
- User Registration & Login
- Secure CRUD operations for blog posts
- Responsive UI
- Protected routes (only authors can edit/delete their posts)

## Setup
1. Clone the repo
2. Backend: `cd backend && npm install && npm start`
3. Frontend: `cd frontend && npm install && npm run dev`
